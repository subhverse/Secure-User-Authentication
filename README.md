# 🔐 Secure User Authentication

A modern full stack authentication system built using React.js, Node.js, Express.js, MongoDB, JWT, and bcryptjs.

This project provides secure user registration, login authentication, protected routes, password encryption, and a modern responsive dashboard UI.

---

# 🚀 Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing using bcryptjs
- Protected Routes
- Session Persistence
- Logout Functionality
- Responsive Modern UI
- Dark Premium Design
- Role-Based Access
- MongoDB Database Integration
- Error Handling & Validation

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Secure-User-Authentication.git
```

---

# 📦 Install Dependencies

## Frontend

```bash
cd client
npm install
```

## Backend

```bash
cd server
npm install
```

---

# ▶️ Run Project

## Start Backend Server

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file inside the `/server` folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
